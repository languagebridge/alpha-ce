Privacy Policy

LanguageBridge Chrome Extension (Alpha Pilot)

Last Updated: December 31, 2025

Version: 1.0.3 (Alpha)

⚠️ Alpha Pilot Version

This is a simplified privacy policy for the LanguageBridge alpha pilot program. This version has no analytics or data collection - only core translation and text-to-speech features powered by Microsoft Azure.

🔒 Our Privacy Commitment:

LanguageBridge Alpha does NOT collect, store, or transmit any personal data, browsing history, or usage analytics. The only data processing occurs through Microsoft Azure's translation and speech services.

1. What Data We Process

Text You Translate

When you select text and request a translation or text-to-speech:

  • Sent to Microsoft Azure: The selected text is sent to Microsoft Azure Cognitive Services for translation and text-to-speech processing
  • Temporary Processing: Azure processes your text and returns the translation/audio
  • Not Stored by Us: We do not store, log, or retain any text you translate
  • Azure's Privacy: Microsoft Azure may temporarily process your text according to their privacy policy (see Section 4)

Settings & Preferences

The extension stores your preferences locally in your browser:

  • Your chosen language (e.g., Persian, Dari, Spanish)
  • Reading speed preference
  • Toolbar visibility preference

Storage Location: All settings are stored locally using Chrome's chrome.storage.sync API. This data syncs across your Chrome browsers if you're signed into Chrome, but we cannot access it.

2. What We DO NOT Collect (Alpha Version)

Zero Analytics - Zero Tracking

This alpha version has all analytics and tracking completely removed:

  • ❌ No usage statistics
  • ❌ No browsing history
  • ❌ No personal information (names, emails, etc.)
  • ❌ No classroom linking or teacher tracking
  • ❌ No database connections (Supabase disabled)
  • ❌ No third-party analytics (Google Analytics, etc.)

3. How Data is Used

Translation & Text-to-Speech (via Microsoft Azure)

Purpose: To provide real-time translation and audio pronunciation

What's Sent:

  • The text you select on a webpage
  • Your chosen target language (e.g., Persian, Dari)

What Happens:

  1. Text is sent securely (HTTPS) to Microsoft Azure Cognitive Services
  2. Azure translates the text and/or converts it to speech
  3. The translation/audio is returned to your browser
  4. The extension displays or plays the result

Retention: We do not retain any of this data. Microsoft Azure's data handling is governed by their privacy policy (see Section 4).

Local Caching (Browser Only)

To improve performance and reduce API calls:

  • Recent translations are cached in your browser's local storage
  • Academic vocabulary lookups are cached for 24 hours
  • All cached data stays on your device and is never transmitted to our servers
  • You can clear this cache by clearing your browser data

4. Third-Party Services

Microsoft Azure Cognitive Services

We use Microsoft Azure for:

  • Translation: Azure Translator Text API
  • Text-to-Speech: Azure Speech Services
  • Speech Recognition: Azure Speech-to-Text (when using voice features)

Azure's Privacy Policy: https://privacy.microsoft.com/en-us/privacystatement

Data Processing: Azure processes text on Microsoft's servers located in the United States (East US region). Azure's data handling is governed by:

  • Microsoft Azure Data Processing Agreement
  • FERPA compliance (for educational institutions)
  • GDPR compliance (for EU users)

5. Data Security

  • Encryption in Transit: All API calls to Azure use HTTPS/TLS encryption
  • No Local Storage of Sensitive Data: We don't store any sensitive content locally
  • API Key Security: Azure API keys are stored securely on our server (Netlify Functions), never in the extension code visible to websites
  • No Database: This alpha version has no database connection, so there's no risk of data breaches

6. Your Rights & Control

What You Can Do

  • Disable the Extension: Disable or remove LanguageBridge from chrome://extensions at any time
  • Clear Local Data: Clear cached translations by clearing Chrome's browsing data
  • Change Settings: Adjust language, speed, and other preferences in the extension popup
  • Limit Usage: Only translate text when you need it - the extension doesn't process anything automatically

7. Children's Privacy (COPPA Compliance)

LanguageBridge is designed for English language learners, including students under 13 years old.

COPPA Compliance (Alpha Pilot):

  • ✅ No personal information is collected from users of any age
  • ✅ No user accounts or registration required
  • ✅ No analytics or tracking of student behavior
  • ✅ All processing is done by Microsoft Azure (COPPA-compliant)
  • ✅ Parents can remove the extension at any time

8. Educational Use (FERPA Compliance)

For schools and teachers using LanguageBridge:

  • No Student Records: We don't collect or store any student educational records
  • No Classroom Data: The alpha version has no classroom linking or teacher dashboards
  • Azure FERPA Compliance: Microsoft Azure is FERPA-compliant for educational institutions
  • School Control: Schools retain full control - they can install, monitor, or remove the extension

9. Changes to This Policy

This is an alpha pilot privacy policy. When we release the full version:

  • We may add optional analytics (with user consent)
  • We may add classroom linking features (with explicit permission)
  • We will update this policy and notify users before collecting any new data

Current Policy Effective Date: December 31, 2025
